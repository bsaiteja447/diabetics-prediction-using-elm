# 🩺 Diabetes Prediction Using Extreme Learning Machine (ELM)

A Machine Learning desktop application that predicts diabetes using patient healthcare data. The system applies data preprocessing, Principal Component Analysis (PCA), and multiple Machine Learning algorithms to compare their performance, with **Extreme Learning Machine (ELM)** achieving the highest prediction accuracy.

---

## 📌 Project Overview

This project combines Software Engineering and Machine Learning techniques to develop an intelligent healthcare prediction system. The application allows users to upload healthcare datasets, preprocess the data, train multiple machine learning models, compare their performance, and predict diabetes for new patient records.

---

## 🎯 Features

- Upload healthcare dataset
- Data preprocessing
- Principal Component Analysis (PCA)
- Train multiple Machine Learning models
- Diabetes prediction for new patients
- Accuracy comparison graph
- Software testing and validation
- User-friendly desktop GUI

---

## 🛠 Technologies Used

- Python
- Tkinter
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- PCA
- Extreme Learning Machine (ELM)

---

## 🤖 Machine Learning Algorithms

- K-Nearest Neighbors (KNN)
- Logistic Regression
- Random Forest
- Linear Support Vector Classifier (Linear SVC)
- Multinomial Naïve Bayes
- Extreme Learning Machine (ELM)

---

## 📂 Dataset

The project uses the **Indian Diabetes Dataset**.

The dataset undergoes:

- Missing value handling
- Data preprocessing
- Feature scaling
- Principal Component Analysis (PCA)

---

## 🚀 Project Workflow

1. Upload Healthcare Dataset
2. Data Preprocessing
3. Apply PCA
4. Train Machine Learning Models
5. Compare Model Accuracy
6. Predict Diabetes
7. Software Testing

---

## 📈 Results

The application compares the performance of multiple machine learning algorithms.

Among all the evaluated models, **Extreme Learning Machine (ELM)** achieved the highest prediction accuracy, making it the best-performing model for diabetes prediction.

## ▶️ How to Run

### Clone Repository

```bash
git clone https://github.com/bsaiteja447/diabetics-prediction-using-elm.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python SEMLHI.py
```

or

```text
run.bat
```

> **Note:** This project was originally developed using Python 3.8/3.9 and older versions of TensorFlow and Scikit-learn. It may require an older Python environment to run successfully.
